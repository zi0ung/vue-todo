# vue-todo
==========

## todo list 소개
본 웹앱은 vue.js와 node.js를 사용하여 만든 웹앱 입니다.

## todo list의 구성
1. 오늘의 할일을 입력하고 엔터를 치거나 혹은 입력 버튼을 눌러서 실시간으로 오늘의 할일을 입력합니다.
2. 원하지 않는 리스트를 따로 삭제하는 기능도 있습니다.
3. 모든 할일을 한번에 삭제하는 clearAll 기능도 있습니다.
